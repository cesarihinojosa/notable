# Notable

A vibe coded visual board app to replace commercial SaaS that paywall simple software.

**Try it: [cesarihinojosa.github.io/notable](https://cesarihinojosa.github.io/notable/)**

Infinite canvas with notes, to-do lists, links, images, columns, arrows, and nested boards. Vanilla JS, no build step, no server, no account. Your data stays in your browser (IndexedDB), with optional auto-save to a local file on Chrome/Edge.

## Run it

Open `index.html` via any static server:

```
npx serve .
```

## Shortcuts

`N` note · `T` to-do · `L` link · `I` image · `B` board · `C` column · `A` arrow · `⌘Z` undo · double-click canvas to add a note
